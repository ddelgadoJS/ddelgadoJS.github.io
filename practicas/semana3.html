Regla horizontal
  • hr

    • Nombre: Regla horizontal.

    Propósito: utilizado para definir un salto temático en una página, es decir separar contenido.

    Sintáxis: ‹hr›

    Atributos:

    • align (no soportado en HTML5) especifica el alineamiento del heading. Los posibles valores son:
      • left
      • center
      • right
    • noshade (no soportado en HTML5) especifica si el elemento se renderizará como un color sólido. El valor es: noshade
    • size (no soportado en HTML5) especifica la altura del elemento. El valor es: pixels
    • width (no soportado en HTML5) especifica la altura del elemento. El valor es: pixels
    • Atributos globales
    • Atributos de evento

Hyperlink
  • a

    • Nombre: Hyperlink.

    Propósito: utilizado para enlazar una página web a otra.

    Sintáxis: ‹a›Ejemplo‹/a›

    Atributos:

    • href especifica el URL del link al que va
    • hreflang especifica el idioma del documento enlazado. El valor es: language_code
    • ping especifica URLs a las cuales se harán post requests, usualmente utilizado para tracking. El valor es: list_of_URLs
    • Atributos globales
    • Atributos de evento

    Ejemplo:

    google.com
